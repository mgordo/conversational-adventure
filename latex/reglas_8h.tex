Creación de Reglas Esta función crea una regla y la guarda en un puntero que previamente era NULL. 

Descripción de la estructura:
\begin{DoxyItemize}
\item t es un TIPOREGLA que permitirá saber cuándo hay que verificar una \hyperlink{structREGLA}{REGLA}
\item num\_\-influido es el numero de acciones a realizar
\item tam\_\-datos\_\-i es el numero de bytes de los datos que se pasarán como argumento a las funciones de las acciones
\item influido es un puntero a un array de ids de las ESTRUCTURAS que hay que cambiar
\item c es un array de valores de CAMPO que permitirán saber qué funciones usar para cambiar los valores
\item valor\_\-cam es un puntero a un array de datos que serán los datos que se introducirán en los respectivos campos marcados con anterioridad
\item num\_\-disparador es el numero de condiciones
\item tam\_\-datos\_\-d es el numero de bytes que contienen los resultados esperados en las funciones de la evaluación
\item disp es un array de ids de las ESTRUCTURAS que disparan/validan la regla
\item key es un array de valores de la enumeración CAMPO que marcará con qué funciones de mundo se validarán los datos
\item valor es un puntero a un array de datos con los que validar los campos
\item anterior es un campo que indica el resultado de la última evaluación de la regla
\end{DoxyItemize}
